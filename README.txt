PRO БЛОКНОТ — версия как приложение для iPhone

Это PWA-приложение.
PWA = Progressive Web App.
Перевод: прогрессивное веб-приложение.

Как установить на iPhone:
1. Нужно разместить папку на хостинге с HTTPS: например GitHub Pages, Netlify, Vercel или свой сайт.
2. Открыть ссылку в Safari на iPhone.
3. Нажать кнопку «Поделиться».
4. Выбрать «На экран Домой».
5. На главном экране появится иконка «Блокнот».
6. После этого он будет открываться как отдельное приложение.

Важно:
- Просто открыть HTML-файл из приложения «Файлы» можно, но это не всегда даёт нормальную установку как приложение.
- Для настоящей установки на экран «Домой» лучше использовать HTTPS-ссылку.
- Записи хранятся локально на устройстве. Делайте «Бэкап» регулярно.
- Это не App Store-приложение и не .ipa-файл. Для .ipa нужен Apple Developer/Xcode/TestFlight/App Store.
